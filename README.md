# nlp_learning
 在datawhale学习nlp文本分类的学习经过中书写的代码，希望自己以后可以循环利用

## Task 02：

建立了一个文章分析类，是一个数据分析和可视化的项目。

详细的功能有：

- 文章长度分析
- 类别分布分析
- 字符个数分布分析
- 不同字符在句子中出现的次数
- 统计每类标签中出现次数最多的字符
- 句尾分析



## Task 04：

针对数据集建立了一个FastText分类器，这是一个三层的神经网络，但是不包括论文中的ngram和分层softmax，最终线下得分0.902，线上得分0.902。

具体的过程为：

1. 我们针对这次的匿名数据集，使用`torchtext`进行数据的预处理
2. 搭建了一个最简单的FastText的模型，但它不包含ngram和分层softmax。
3. 构造了训练，验证，预测的整个迭代过程

总体来说，是一个比较完整的简单的项目。
